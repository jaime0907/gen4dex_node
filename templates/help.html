Welcome to the Gen 4 Dex! This web is a free resource to help you in your jorney to catch every Pokémon in Pokémon Diamond/Pearl/Platinum and HeartGold/SoulSilver.

The main page shows a huge table filled with every Pokémon and their location inside the games, along with the level it appears and how rare it is.
In the top you can find the menu bar, which has some filters for the data table.
The first filter lets you choose between showing the best location for each Pokémon, or every possible location where it can be obtained. The "best" one is the the place where the Pokémon appears more frecuently.
The second one lets you choose the number of results the table will show. It is limited to 100 because higher values would affect performance
The third one shows the list of methods, so you can hide some hideous ones like Poké Radar or Swarm.
The next filters are the game ones. You can activate or deactivate them so the data table only shows Pokémon obtaniable in those games.
The Evo, Egg and Event ones are similar, they let you show or hide Pokémon obtainable by evolving, hatching eggs or via events, for those who are just focusing on catching.
The rest are links to the login, register and help pages. If you are logged in, your username will appear instead. Clicking it will take you to your profile page, where you can view and update your Pokédex info.
The big text box lets you filter either by name or by dex number. The number filter is a "greater than" filter, so if you type 120 you'll get every Pokémon avaliable from #120 onwards.
Registration is not needed for using the app, but it lets you save a personal Pokédex, so the Pokémon you already caught won't appear on the data table.
In order to register a Pokémon you can click its name on the data table, or register it on your profile page.
The app stores the passwords hashed and salted, so in the event of a security breach the passwords will be fine.

If you find any bug, or have any suggestion, please let me know.
This webapp doesn't have any ads and it's free to use.
